Dial-up Internet in 2023?

April 13, 2023

Can Dial-up internet still exist in 2023? Apparently, the answer is yes. Last summer I managed to get my Dell Dimension e510, with the help of a VOIP box, to accept incoming connections through its dial-up modem and actually provide an internet connection through it! Now, why would you want to do this in 2023? Dial-up internet is basically useless with the internet of today. But if you're into vintage computing like me, there are several reasons why it can be interesting to set something like this up yourself.

Dialing up!

Loading a 2001 snapshot of Microsoft.com


  • File Transfers: When trying to transfer data between old and new computers, it can be a pain to have to split a file up, and swap floppy disks every 3-4 minutes. Using the modem instead may still be slow, but this at least allows for the process of copying files to be more automated.
  • Vintage Web Browsing: If you load archived web sites from the Internet Archive, you can simulate what it actually was like to use the internet on a dial-up connection back when they were more common.
  • The Sound: It's just neat to listen to in my opinion!
